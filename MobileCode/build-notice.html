Building Mobile Code from Source

Mobile Code is built with Flutter, allowing you to compile it for multiple platforms. However, each platform has specific build requirements.

Prerequisites

  • Flutter SDK 3.0 or higher
  • Git
  • Platform-specific development tools (see below)

Install Flutter

# Download Flutter from https://flutter.dev/docs/get-started/install # Add Flutter to your PATH export PATH="$PATH:`pwd`/flutter/bin"

ü™ü Building for Windows

Requirements: Windows 10 or higher, Visual Studio 2022

# Clone the repository git clone https://github.com/yourusername/mobile-code.git cd mobile-code # Install dependencies flutter pub get # Build for Windows flutter build windows --release # Output will be in build/windows/runner/Release/

Note: Windows builds can only be created on Windows machines.

üêß Building for Linux

Requirements: Ubuntu 20.04 or higher, build-essential, cmake

# Install Linux dependencies sudo apt-get update sudo apt-get install clang cmake ninja-build pkg-config libgtk-3-dev # Clone the repository git clone https://github.com/yourusername/mobile-code.git cd mobile-code # Install dependencies flutter pub get # Build for Linux flutter build linux --release # Output will be in build/linux/x64/release/bundle/

Note: Linux builds can only be created on Linux machines.

üçé Building for macOS

Requirements: macOS 10.15 or higher, Xcode 14+

# Clone the repository git clone https://github.com/yourusername/mobile-code.git cd mobile-code # Install dependencies flutter pub get # Build for macOS flutter build macos --release # Output will be in build/macos/Build/Products/Release/Mobile Code.app

Note: macOS builds can only be created on macOS machines.

iOS

# Requires macOS with Xcode flutter build ios --release

Android

# Works on any platform flutter build apk --release flutter build appbundle --release

Development Setup

For development and testing:

# Run in debug mode flutter run # Run on specific device flutter run -d macos flutter run -d windows flutter run -d linux # Run tests flutter test # Analyze code flutter analyze

Troubleshooting

  • Flutter not found: Make sure Flutter is in your PATH
  • Platform not supported: You need to be on the target platform to build for it
  • Build errors: Run flutter doctor to check your setup
  • Dependencies issues: Try flutter clean && flutter pub get
